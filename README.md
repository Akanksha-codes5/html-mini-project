# Mini Web Project

## Description
This mini web project is developed using **HTML, CSS, and JavaScript** to demonstrate the basics of front-end web development. The project consists of multiple interconnected web pages with a navigation menu and showcases various web features such as lists, forms, arithmetic operations, image galleries, and multimedia content.

HTML is used to structure the web pages, CSS is used for styling and layout, and JavaScript is used to add interactivity and basic functionality such as calculations and form validation.

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  

## Project Modules
- Home Page  
- Mobile Devices List  
- Student Records  
- Arithmetic Operations  
- Image Gallery  
- Forms  
- Multimedia  

## Objective
The objective of this project is to understand and apply fundamental web development concepts and to build a simple, interactive website using front-end technologies.

## Conclusion
This project provides hands-on experience with creating a basic website and helps in understanding how HTML, CSS, and JavaScript work together to build interactive web applications.
